# swiftui-movie-search-app

SwiftUI Movie Search App

### Summary

- Search and list movies by keyword
- Selecting a movie from the list and seeing its details

### Includes

- Generic Network Layer
- MVVM Design Pattern
- OMDb API
- URLSession
- Clean Folder Organization
